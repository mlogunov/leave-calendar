## Minimal Path to Awesome

- Clone this repository
- In write-manifests.json fill in the correct CDN path
- In the command line run:
  - `npm install`
  - `gulp build`
  - `gulp bundle --ship`
  - `gulp package-solution --ship`
  - `Add to AppCatalog and deploy`
